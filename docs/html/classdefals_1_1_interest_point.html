Detailed Description

This class acts like a wrapper for cv::KeyPoint. It associated a KeyPoint with its descriptor instead of having to find it as a row of a matrix.

The matching algorithm needs to consider two lists of InterestPoint:

  • one ordered by the keypoint's angle ;
  • the other ordered by the keypoint's descriptor's norm. However, we need to be able to find efficiently where is one keypoint in both lists. To do so, we provide an InterestPoint with two attributes standing for their index in both lists.
