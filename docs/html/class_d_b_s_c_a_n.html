Detailed Description

This class stands for a scanner using DBSCAN algorithm. It specializes in the detection of dense clusters of segments represented in 4D by: (x, y, theta, l) where:

  • (x,y) : closest end of the segment to (0, 0)
  • theta : the polar angle of the segment
  • l : the length of the segment
